這版標註原則

棕色：未標註高優先候選 灰色：一般可標註路段 綠色：已標註 紅色：目前選取 藍色：目標路段 紅點：目前路段附近路口
  1. OSM 有的車道數、turn:lanes、motorcycle tag 先當底，不重標。
  2. 人工主要補：二段式左轉牌、待轉區、待轉區位置，以及 OSM 機車限制是否代表二段式。
  3. forward 是 OSM way 起點到終點；下方會顯示起點/終點與大概往哪個方位。

這裡是標「目前 A 路段沿某個 OSM 方向進入此路口」的規則。另一條路或對向入口請切換到那條路段再標。

車道配置
